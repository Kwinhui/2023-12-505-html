# 와이어 프레임
1. 와이어프레임(wireframe)의 개념
- 웹사이트의 회로도(schematic)또는 청사진(blueprint)라고도 불리우며, 웹사이트의 골격을 나타내는 시각적 가이드
2. 와이어프레임(wireframe)의 구성요소
- Wrap : 전체박스
- Header : 헤더
- Slide : 이미지 슬라이드
- Contents : 콘텐츠
- Footer : 푸터
- Tab Menu : 탭메뉴
- Pop-up : 팝업창
3. 와이어프레임(wirefream)의 종류
- 가로형과 세로형이 있음
